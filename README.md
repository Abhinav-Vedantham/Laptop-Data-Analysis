# 💻 Laptop Data Analysis

## 📌 Overview
This project analyzes a dataset of laptops to understand how different features, such as RAM, processor, storage, and brand, influence laptop pricing.

---

## 🎯 Objectives
- Analyze the price distribution of laptops
- Compare pricing across brands
- Study the relationship between performance features and price
- Categorize laptops into budget, mid-range, and premium segments

---

## 📊 Dataset
- Source: Kaggle Laptop Dataset
- Features include:
  - Brand
  - Processor
  - RAM
  - Storage
  - Price

---

## ⚙️ Data Preprocessing
- Created new columns:
  - Price Category (Budget, Mid-range, Premium)
  - CPU Type
  - Storage Type
- Used grouping and aggregation for analysis

---

## 📈 Visualizations
- Brand vs Average Price (Bar Chart)
- Price Distribution (Histogram)
- Price Variation Across Brands (Box Plot)
- RAM vs Price (Scatter Plot)
- CPU Type vs Price (Box Plot)

---

## 🔍 Key Insights
- Higher RAM leads to a higher price
- Premium laptops have better specs
- Processor type significantly affects pricing
- Brands differ in pricing strategy
